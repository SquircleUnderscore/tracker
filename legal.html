Mentions légales et Politique de confidentialité

Dernière mise à jour : 23 novembre 2025

1. Éditeur du site

Habit Tracker est un service fourni à titre personnel.
Site web : https://tracker.squircle.computer
Hébergement : Cloudflare Pages

2. Collecte et traitement des données

2.1 Données collectées

Nous collectons uniquement les données suivantes :

  • Adresse email : utilisée uniquement pour l'authentification et la connexion à votre compte
  • Données d'habitudes : vos tâches, icônes, et états de progression (stockées localement et dans le cloud si vous êtes connecté)
  • Informations d'authentification OAuth : lors de la connexion via GitHub ou Google

2.2 Base légale du traitement

Le traitement de vos données repose sur :

  • Votre consentement lors de la création de compte
  • L'exécution du contrat de service

2.3 Durée de conservation

Vos données sont conservées tant que votre compte est actif. Vous pouvez les supprimer à tout moment via le menu "Mon compte".

3. Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès : vous pouvez consulter vos données à tout moment dans l'application
  • Droit de rectification : vous pouvez modifier vos tâches et données directement dans l'application
  • Droit à l'effacement : vous pouvez supprimer votre compte et toutes vos données via "Mon compte" → "Supprimer mon compte"
  • Droit à la portabilité : vous pouvez exporter toutes vos données au format JSON via "Mon compte" → "Exporter mes données"
  • Droit d'opposition : vous pouvez vous opposer au traitement en supprimant votre compte

4. Sécurité des données

Nous mettons en œuvre des mesures de sécurité appropriées :

  • Connexions HTTPS chiffrées
  • Authentification sécurisée via Supabase (OAuth2, Magic Links)
  • Base de données sécurisée avec contrôle d'accès RLS (Row Level Security)
  • Stockage local chiffré par le navigateur

5. Cookies et technologies similaires

L'application utilise :

  • LocalStorage : pour stocker vos données localement (persistance hors ligne)
  • Cookies d'authentification : pour maintenir votre session (Supabase Auth)
  • Service Worker : pour le fonctionnement hors ligne (PWA)

Aucun cookie tiers de tracking ou publicité n'est utilisé.

6. Partage des données

Vos données ne sont jamais vendues ni partagées avec des tiers, sauf :

  • Supabase (hébergement base de données, certifié SOC 2 Type II)
  • Cloudflare (hébergement site web et protection DDoS)
  • Fournisseurs OAuth (GitHub, Google) uniquement pour l'authentification

7. Conditions générales d'utilisation

7.1 Objet

Habit Tracker est une application gratuite de suivi d'habitudes. Le service est fourni "tel quel" sans garantie.

7.2 Responsabilités

L'utilisateur est responsable de :

  • La sauvegarde régulière de ses données (export JSON recommandé)
  • La sécurité de son compte (mot de passe, email)
  • L'utilisation conforme du service

7.3 Disponibilité

Nous nous efforçons d'assurer une disponibilité maximale du service, mais ne pouvons garantir une disponibilité 24/7. Le mode hors ligne permet de continuer à utiliser l'application sans connexion.

7.4 Modification du service

Nous nous réservons le droit de modifier ou d'interrompre le service à tout moment, avec préavis si possible.

8. Contact

Pour toute question concernant vos données personnelles ou ces mentions légales, vous pouvez nous contacter via GitHub : github.com/SquircleUnderscore/tracker

9. Modifications

Nous nous réservons le droit de modifier cette politique de confidentialité à tout moment. Les modifications seront publiées sur cette page avec une nouvelle date de mise à jour.
